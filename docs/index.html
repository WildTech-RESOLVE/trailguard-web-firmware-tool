Install the Latest TrailGuard Firmware

  1. Connect the Communications Unit to your computer with a Micro-USB data cable. Insert the setup magnet into the slot next to the camera to activate Setup Mode. Both SIG.1 and SIG.2 LEDs should illuminate.
  2. Click "Install Firmware" and select the serial port (look for 'USB JTAG/serial...' or 'COM1'). Toggle "Factory Reset" to erase all data before installation.
  3. Open "Advanced Options" to install a specific firmware version, erase the device manually, reboot after installation, or view the logs console.
  4. Wait for the status bar to confirm completion before unplugging the device.

Before you begin:

        

        
    
          
  • Confirm the power cable is connected and the SIM card is properly seated.
    • 
          
  • Check that both antennas are securely attached to the Communications Unit.
    • 
          
  • Ensure the motion sensor is covered and all indicator lights are off.
    • 
          
  • If you are testing normal operation, keep the magnet out of the slot.
