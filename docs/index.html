Install the Latest TrailGuard Firmware

  1. Connect the Communications Unit to your computer with a Micro-USB data cable. Insert the setup magnet into the slot next to the camera to activate Setup Mode. Both SIG.1 and SIG.2 LEDs should illuminate.
  2. Click the "Install" button below, grant Web Serial permissions, and select the correct serial port (it should look like 'USB JTAG/serial...').
  3. Tick “Erase device” if you want to reset all saved settings and statistics to factory defaults.
  4. Monitor the progress dialog and wait for the flashing process to finish. Do not disconnect until the success message appears.
  5. After the update completes, confirm the camera reboots and re-enters Setup Mode. Remove the setup magnet and trigger the camera again to enter normal operation.

Tips:

  • Please use Google Chrome or Microsoft Edge on a desktop computer for Web Serial compatibility.
  • Keep this tab open and leave the device connected until installation completes.
  • If the browser cannot find the device, check if the USB cable is a data cable, re-seat the USB cable and confirm the camera remains in Setup Mode.
  • Close other serial consoles before connecting.
  • Approve any Web Serial permission prompts in Chrome/Edge; the tool requires serial access to flash.
